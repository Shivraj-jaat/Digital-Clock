# ⏱️ Digital Clock

A modern and responsive **Digital Clock** built using **HTML, CSS, and JavaScript**. The application displays the current time, date, day, and supports **Dark/Light Mode** along with **12-hour / 24-hour time format**.

## 🚀 Features

* 🕒 Live Digital Clock
* 📅 Current Day, Date, Month & Year
* 🌗 Dark / Light Mode Toggle
* ⏳ 12-Hour / 24-Hour Time Format
* 📱 Responsive Design
* 🎨 Modern Gradient UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
